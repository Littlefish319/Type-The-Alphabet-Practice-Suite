# Alphabet Typing Suite

Alphabet Typing Suite is a practice + analytics tool for typing the alphabet faster.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set `VITE_GEMINI_API_KEY` in `.env.local` to your Gemini API key (optional, only needed for the AI Coach)
3. Run the app:
   `npm run dev`
